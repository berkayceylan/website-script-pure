
# Original CMS System

CMS on Pure PHP, Mysql backend and HTML, CSS, Javascript, Jquery Frontend



## Features
 ### User
- Sign Up, Login,
- Change your profile
- Upload profile photo
 ### Blog
- Add new post from Admin panel
- Add featured image to the post
- Edit and delete
 ### Admin Panel
- Add and edit post
- See the messages
- See and edit users
 ### Others
- Comments System
- Messages System
- Last Posts
